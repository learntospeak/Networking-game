HTTP Request

Ask

The browser creates `GET /profile` to ask the server for the page.

Travel

The request crosses the network path until the web server receives it.

HTTP vs HTTPS

HTTP

The request stays readable while it travels, so `GET /profile` is visible on the path.

HTTPS

The same request is carried inside encrypted traffic, so the path only sees a locked packet.

Server Response

200 OK

If the page exists, the server sends back `200 OK` with the page content.

Load

The browser uses the response to render the page for the user.
